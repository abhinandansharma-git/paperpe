# PaperPe - Professional Landing Page

A modern, production-ready landing page for PaperPe - the paper trading platform for algorithmic traders.

## 🎨 Design Features

- **Professional Fintech Aesthetic**: Inspired by Stripe, Linear, and Vercel
- **Color Scheme**: Deep navy (#0F172A) with orange accents (#F97316)
- **Typography**: Inter font with proper hierarchy
- **Icons**: SVG icons from Lucide React (no emojis)
- **Animations**: Smooth 200-300ms transitions, subtle float effects
- **Glassmorphism**: Real backdrop-blur effects with proper contrast
- **Mobile-First**: Fully responsive design

## 🏗️ Tech Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Lucide React

## 📦 Installation

```bash
npm install
```

## 🚀 Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## 🏭 Production Build

```bash
npm run build
npm start
```

## 📄 Sections

1. **Hero** - Bold headline with email capture
2. **Social Proof** - Key metrics (500+ traders, ₹10L capital, 100% free)
3. **Features** - 4 feature cards with hover effects
4. **Code Example** - Syntax-highlighted API showcase
5. **Pricing** - Free vs Pro comparison
6. **Final CTA** - Email capture with decorative background

## 🎯 Key Design Principles

- **Clean & Confident**: No clutter, generous whitespace
- **Trustworthy**: Professional color palette and typography
- **Engaging**: Smooth hover effects and transitions
- **Accessible**: Proper contrast ratios and semantic HTML
- **Performance**: Optimized build, minimal JavaScript

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Desktop: ≥ 768px
