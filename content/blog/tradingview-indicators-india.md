## How to Add Indicators in TradingView

1. Open TradingView chart
2. Click Pine Editor (bottom tab)
3. Delete default code
4. Paste indicator code
5. Click Add to Chart
6. Done!

- **EMAs**: 9, 21, 50 work well
- **RSI**: 14 period, 30/70 levels
